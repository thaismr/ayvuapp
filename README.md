# Ayvuapp

## Project setup

### Custom users app:

[Docs: startapp](https://docs.djangoproject.com/en/4.0/ref/django-admin/#startapp)

````shell
py manage.py startapp ayvuapp.users ./ayvuapp/users
````

### Style formatting

<https://github.com/google/yapf/>

```shell
yapf -r -i . --style google
```
